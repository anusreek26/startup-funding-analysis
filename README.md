# Startup Funding Analysis

## Project Overview

This project analyzes startup funding trends in India using Python to identify funding patterns, leading investors, top-funded cities, industries, and investment types.

The project demonstrates data cleaning, exploratory data analysis (EDA), visualization, and business insight generation using real-world startup funding data.

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## Dataset

Startup Funding Dataset containing information such as:

- Startup Name
- City
- Industry
- Investors
- Investment Type
- Funding Amount
- Date

---

## Project Workflow

1. Data Cleaning
2. Data Preprocessing
3. Exploratory Data Analysis
4. Data Visualization
5. Business Insights

---

## Key Insights

- Identified the cities receiving the highest startup funding.
- Found the industries attracting the most investments.
- Analyzed yearly funding trends.
- Identified top investors.
- Compared different investment types.
